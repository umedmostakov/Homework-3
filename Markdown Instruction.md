# Инструкция для работы с Markdow

## Выделение текста

 Мы можем выделить текст курсивом обрамив его звездочками (*).
 
Пример : *Вот так*

 Чтобы выделить текст полужирным нам необходимо обрамить его 2 звездочками (*).

Например : **Вот так**
